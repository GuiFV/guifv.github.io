IDETRA

Plataforma educacional + sistema de ger. de projetos

Este é um projeto pessoal meu. Criei esta plataforma para absorver voluntários dispostos a trabalhar por uma boa causa. Desde sua concepção esta era uma plataforma muito complexa de ser desenvolvida, pois precisava que abrangesse a explicação do que é a iniciativa IDETRA, uma plataforma de educação e um sistema de gestão de projetos. Construída completamente do zero, foi utilizado bootstrap no front end e Django no backend. Codifiquei todo o front-end e a maior parte do back-end principal. Eventualmente, tive ajuda para terminar algumas das funcionalidades mais complexas.
