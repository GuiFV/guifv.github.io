Renomeador de arquivos

Meu primeiro script em Python

Eu tinha um problema: toda vez que minha Nikon D90 tinha mais de 9999 fotos no cartão SD, ela abria uma nova pasta e colocar as novas fotos lá a partir de 0000. E eu tinha muito mais do que isso por trabalho tirando fotos para uma agência. Eu tinha que agilizar a renomeação dos arquivos para mais de 9999 para que eles pudessem caber em uma única pasta. E renomear arquivos é algo que computadores devem fazer. Então, criei um script simples que pergunta qual caracter você deseja substituir, em que posição (caso seja repetido), e substituir por qual. Agora minhas fotos podem começar na posição 99.999 ou mais! E você pode usar este script para qualquer arquivo que desejar. Lembre-se: este foi meu primeiro caso de uso real de aplicativo Python
