File renamer

My first Python script

I had a problem: every time my D90 Nikon had over 9999 pictures in the SD card, it would open a new folder and place new pictures there starting from 0000. And I had much more than that per job taking pictures for an agency. I had to streamline renaming the files to over 9999 so they could fit in one folder. And renaming files is something computer should do. So I created a simple script that asks what character you want to replace, in what position (in case it repeats itself), and replace it for what. Now my pictures can start at the position 99.999 or more! And you can use it to any files you want. Just remember: this was my very first real use case Python application.
